CowinVue Help
  • The availability of the slots can be searched using Pincode or Districts. Multiple Pincodes (each Pincode separated by ",") and District can be selected while searching for slots
  • Select State, provide Pincodes/ select districts from the dropdown, and selected required filters from the available filter options. (Filters can be set before or after the search)
  • Click on "Start Search" button to initiate an auto search
  • If slots are available in any selected Districts/ Pincodes, the same will be listed with an alert. Details of each slot can be viewed by clicking on the listed slot card
  • If no available slots are found, the search will be auto-triggered based on the selected interval. The default interval is 30 Seconds
  • The search can be cancelled by clicking on the "Cancel Search" button
  • This app uses CoWin open API to make it easy for you to find slots. Availability changes in real-time. So book your slot ASAP
